A simple Rock-Paper-Scissors game built using Python and Gradio where users can play against the computer through a web interface.

🚀 Features

Play against the computer

Random computer choice

Real-time result display

Score tracking for user and computer

🛠 Technologies Used

Python

Gradio

Random module

▶️ How to Run

Install the required library:

pip install gradio

Run the Python script or notebook.

A local web interface will open where you can select Rock, Paper, or Scissors to play against the computer.

📂 Project File

rock_paper_scissors.py or rock_paper_scissors.ipynb – Main program file
